Flexbox - Propiedad Order

Lo que hace la propiedad order de CSS cuando se está usando Flexbox es el reorganizar los elementos hijo que pertenecen a un contenedor padre flexible (display flexbox). Si bien se puede cambiar el orden de los elementos desde el código fuente, CSS también lo permite hacer sin tocar el HTML. Por defecto, la propiedad order estará establecida en 0 para todos los elementos hijo dentro de un flexbox, pero puede modificarse ese valor para así afectar el orden en que se visualiza.

La propiedad acepta valores numéricos tanto positivos (El valor 0 siempre será visualizado como el primer elemento del flexbox) como negativos (Entre más este el valor lejos de 0, se visualizará como el primer elemento del flexbox sin importar si hay otros con valor de 0).

Si hay elementos con un valor order idénticos, se visualizará en el mismo órden en que aparece en el código fuente.
