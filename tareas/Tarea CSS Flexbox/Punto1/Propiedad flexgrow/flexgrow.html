Flexbox - Propiedad Flexgrow

La propiedad flexgrow lo que hace es darle un ancho a los elementos hijo de un contenedor flexible en proporción a cuántos elementos estén en el y el número dado a cada elemento hijo. Esto quiere decir que con flexgrow no necesitamos usar width en los elementos hijo, pues esta propiedad se encargará de darle ese width según el viewport width del dispositivo en que se está visualizando.

La forma en como se le especifica el valor a esta propiedad es mediante números enteros iniciando desde 0. Entre mayor sea el número, más ancho ocupará dentro del contenedor padre. El valor por defecto es 0, eso significa que es como si la propiedad estuviera en ninguno o none y por tanto, los elementos hijo sólo ocuparán el ancho y alto necesarios.

Cabe destacar que el contenedor padre deber tener la propiedad display: flex y que flexgrow se le asigna a cada uno de los elementos hijo. El elemento hijo que tenga el número más alto en flexgrow será el que ocupará más espacio dentro del contenedor y lo mismo sucede con el elemento hijo con el número más bajo pero al contrario, ocupará menos espacio.
