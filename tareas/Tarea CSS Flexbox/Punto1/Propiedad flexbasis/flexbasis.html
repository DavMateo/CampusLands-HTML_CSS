Flexbox - Propiedad Flexgrow

La propiedad flex-basis permite establecer un ancho base a un elemento hijo que se encuentra en un contenedor flexible. Esto permite establecer un ancho personalizado máximo y que se irá adaptando al viewport width del dispositivo, esto quiere decir que el elemento hijo tratará de tomar el ancho especificado en flex-basis en caso de que la pantalla no permita mostrarlo por completo.

La propiedad flex-basis recibe valores con unidades de medidas ya sean relativas o absolutas. Básicamente es similar a emplear la propiedad width para manejar el ancho del elemento, solo que con flex-basis no permitirá que se desborden los elementos cercanos o se dañe la visualización de la web.

El contenedor padre debe llevar la propiedad display: flex y los elementos hijo contendrán la propiedad flex-basis, de lo contrario, no surtirá efecto esta característica.
